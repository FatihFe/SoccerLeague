# SoccerLeague
-This application shows the soccer team list on the main screen. When the draw fixture button is clicked on the main screen, weekly team matches appear.

# Features 
- MVVM 

- OOP 

- JAVA 

- Room 

- LiveData 

- ViewModel 

- Dark Mode

- JSON 

- Signed APK
